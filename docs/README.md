## 项目定位

这个 App 不是传统“背单词”，而是“快速刷词”：

- 用户只做 `认识 / 不认识` 的快速判断
- 不认识自动进入错词本
- 错词继续回刷
- 词书支持手动添加和批量导入
- 最终支持电脑和手机同步
